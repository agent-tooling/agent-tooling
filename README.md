# agent-tooling

`github.com/agent-tooling` contains useful utilities and packages for agent development.
It is intended to support building, integrating, and running agent runtimes and related tooling.
These components are designed to make practical agent workflows easier to develop and maintain.

This repository itself is not the full org mirror. It includes root-level agent instructions for developing and maintaining the packages and repositories within `agent-tooling/`.

## Initialize Local Mirror

Use the `agent-tooling` skill to initialize `agent-tooling/`:

1. Create an `agent-tooling/` folder that acts as the GitHub org mirror.
2. Clone `git@github.com:agent-tooling/agent-tooling.git` into `agent-tooling/agent-tooling/`.
3. Copy `AGENTS.md` and `.agents/` from `agent-tooling/agent-tooling/` into the root `agent-tooling/` folder.
4. Tell the agent to initialize `agent-tooling/` by following the `agent-tooling` skill.

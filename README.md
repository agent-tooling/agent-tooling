# agent-tooling

`github.com/agent-tooling` contains useful utilities and packages for agent development.

This repository includes root-level agent instructions for developing and maintaining the packages and repositories within `agent-tooling/`.

## Initialize Local Mirror

1. Create an `agent-tooling/` folder that acts as the GitHub org mirror.
2. Clone `git@github.com:agent-tooling/agent-tooling.git` into `agent-tooling/agent-tooling/`.
3. Copy `AGENTS.md` and `.agents/` from `agent-tooling/agent-tooling/` into the root `agent-tooling/` folder.
4. Tell your agent to initialize `agent-tooling/` root folder by following the `agent-tooling` skill.
